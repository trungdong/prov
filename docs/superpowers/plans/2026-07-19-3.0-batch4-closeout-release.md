**Process rules:** as prior batches — one task = one PR = green CI = merge commit; refactor tasks are **behaviour-preserving** (suite counts unchanged, no serialized-output change); verify with `uv run pytest -q`, `uv run ruff check src/`, `uv run ruff format --check src/`, `uv run mypy src`; complexity claims proven with `uvx lizard -C 15 <file>`; CI watch via `gh run list`/`gh run watch`; Codacy advisory.
