About the Provenance Server

{% endblock %} {% block content %}

The Provenance Server is a web service that allows you to store, browse and manage your provenance bundles. The server can be accessed via this Web interface or via the REST API (using an API key or OAuth authentication).

Technology

This service was built upon the ProvPy package, which provides an Python implementation of the PROV specifications being standardised at the W3C Prov Working Group.
The web server is built on Django Web framework.

Acknowledgements

The ProvPy package and this service are funded by an EPSRC grant (EP/H500243/1) and by the ORCHID Programme.
